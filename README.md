# Uber Ride Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard analyzing Uber ride bookings data to generate business insights.

## Objectives
- Analyze ride demand trends
- Monitor booking performance
- Identify peak booking periods

## Tools Used
- Power BI
- Data Analysis
- Data Visualization

## Key Features
- Interactive dashboard
- KPI indicators
- Dynamic filters
